# C-Game
Um compilado de minijogos desenvolvidos totalmente na linguagem **C**. 
Criado por [@DRibeiroSs](https://github.com/DRibeiroSs) e [@vichugo-bit](https://github.com/vichugo-bit).

---

## Jogos Inclusos

* **Perguntas e Respostas:** O jogador deve responder a 5 perguntas corretamente para acumular pontos. 
    * *Jogadores:* 1 jogador.

* **Cobra na Caixa:** Um jogo de sorte e azar com 5 caixas randômicas. Ao abrir uma caixa, você pode ganhar, perder ou nada acontecer.
    * *Jogadores:* 2 jogadores.

* **Gousmas War:** Um jogo altamente imprevisível onde os jogadores se enfrentam.
    * *Jogadores:* 2 jogadores.

---

## Como compilar e executar

Como este projeto é feito em C, você precisará de um compilador (como o GCC) ou um IDE (como o DevC++) instalado na sua máquina.







# English translated


# C-Game
A collection of minigames developed entirely in the **C** programming language.
Created by [@DRibeiroSs](https://github.com/DRibeiroSs) and [@vichugo-bit](https://github.com/vichugo-bit).

---

## Included Games

* **Questions and Answers:** The player must answer 5 questions correctly to earn points. 
    * *Players:* 1 player.

* **Snake in the Box:** A game of chance featuring 5 random boxes. When opening a box, you can win, lose, or nothing might happen at all.
    * *Players:* 2 players.

* **Gousmas War:** A highly unpredictable game where players face off against each other.
    * *Players:* 2 players.

---

## How to compile and run

Since this project is written in C, you will need a compiler (such as GCC) or an IDE (such as Dev-C++) installed on your machine.
